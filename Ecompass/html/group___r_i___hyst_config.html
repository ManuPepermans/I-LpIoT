Enable or disable Hysteresis of the input schmitt triger of Ports A When the I/Os are programmed in input mode by standard I/O port registers, the Schmitt trigger and the hysteresis are enabled by default. When hysteresis is disabled, it is possible to read the corresponding port with a trigger level of VDDIO/2.

Parameters
__IOPIN__: Selects the pin(s) on which to enable or disable hysteresis. This parameter must be a value of PIN define
Return values
None
