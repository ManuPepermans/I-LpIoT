#define __HAL_FLASH_ACC64_DISABLE ( )    (CLEAR_BIT((FLASH->ACR), FLASH_ACR_ACC64))

Disable the FLASH 64-bit access.

Note
Read access 32 bit is used
To reset this bit, the LATENCY should be zero wait state and the prefetch off.
Return values
none
