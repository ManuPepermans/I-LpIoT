◆ IS_UART_ADDRESS

#define IS_UART_ADDRESS (   __ADDRESS__)    ((__ADDRESS__) <= 0xF)

Check UART Node Address ADDRESS: UART Node address specified by the user UART Node address is used in Multi processor communication for wakeup with address mark detection. This parameter must be a number between Min_Data = 0 and Max_Data = 15 Return : TRUE or FALSE

◆ IS_UART_BAUDRATE

#define IS_UART_BAUDRATE (   __BAUDRATE__)    ((__BAUDRATE__) < 4000001)

Check UART Baud rate BAUDRATE: Baudrate specified by the user The maximum Baud Rate is derived from the maximum clock on APB (i.e. 32 MHz) divided by the smallest oversampling used on the USART (i.e. 8) Return : TRUE or FALSE
