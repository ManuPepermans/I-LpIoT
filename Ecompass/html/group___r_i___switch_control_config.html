◆ __HAL_RI_SWITCHCONTROLMODE_ENABLE

#define __HAL_RI_SWITCHCONTROLMODE_ENABLE ( )    SET_BIT(RI->ASCR1, RI_ASCR1_SCM)

Enable or disable the switch control mode.

Note
ENABLE: ADC analog switches closed if the corresponding I/O switch is also closed. When using COMP1, switch control mode must be enabled.
DISABLE: ADC analog switches open or controlled by the ADC interface. When using the ADC for acquisition, switch control mode must be disabled.
COMP1 comparator and ADC cannot be used at the same time since they share the ADC switch matrix.
Return values
None
